### Node version
- Required: 8.0.0 -> 12.19.0

### How to run
- Install
```bash
yarn
```

- Start
```bash
yarn start
```

- Unit test with Jest + Enzyme
```bash
yarn test
```

- Build
```bash
yarn build
```
